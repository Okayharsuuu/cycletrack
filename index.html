📡 Push WiFi Credentials

Send WPA2-Enterprise (college WiFi) credentials to a device via Firebase. The ESP32 reads and applies them on next boot.

How it works: Credentials are written to /devices/{id}/config in Firebase. The ESP32 reads them on boot, connects, then deletes the password from Firebase for security. The "Config pending" badge in the table clears once applied.
